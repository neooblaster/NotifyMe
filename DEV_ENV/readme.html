Console: 


		
		
		
		

			I. Initialize NotifyMe
		

		
		

			Firt, you can create a new variable which will be a new instance of NotifyMe()
		

		
		

		var notifier = new NotifyMe();
		

		
		
		
		

			II. Define Notification Title
		

		
		

			To send a notification, you muse define a title :
		

		
		

		notifier.title('Notification Title Here');
		

		
		
		
		

			III. Define The Message (Body)
		

		
		

			To send a notification, you muse define a messsage :
		

		
		

		notifier.message('Notification message also call body');
		

		
		
		
		

			IV. Notification Options
		

		
		

			You can send some options to configure your notification. To set options, just use the dedicated method :
		

		
		

			You can send a JSON object, or a String which respect the JSON Syntax.
		

		
		

			To learn more about options, you can use the embed method help !
		

		
		

		notifier.options('{icon: "http://demo.neoblaster.fr/Files/Web-icon.png"}');
		notifier.help();
		

		
		
		
		
		

			V. Define the display duration
		

		
		

			You can custom the display duration of the notification with the close method. The parameter is the delay in milliseconds.
		

		
		

		notifier.close(3000);
		

		
		
		
		
		
		
		
		

			VI. Send Notification
		

		
		

			When all parameters are setted, you can send the notification. You you never trigger a notification, first, you will have a request to allow notifications.
		

		
		

		notifier.send();
		

		
		
		
		

			VII. Request Authorization without send a notification
		

		
		

			If you want request an authorization without send a notification, you can with the request method :
		

		
		

		notifier.request();
		

		
		
		
		

			VIII. Successive methods
		

		
		

			The main interest of the JavaScript class it to call all methods consecutivly like that :
		

		
		

		new NotifyMe().title('Notif Title - Consecutive call').message('Notif Message - Consecutive call').options({icon: 'http://demo.neoblaster.fr/Files/Web-icon.png'}).close(3000).send();
